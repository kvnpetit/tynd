# CLI Reference

```bash
bunx @tynd/cli <command> [options]
# or (after installing)
tynd <command> [options]
```

## Global flags

| Flag | Effect |
|---|---|
| `--verbose` | Debug-level logs — IPC traffic, cache decisions, Rust-side events |
| `--help` | Show help for a command |
| `--version` | Print CLI version |
